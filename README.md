# Neon Dash Rescue

A fun first GitHub project: a small 2D arcade game built with plain HTML, CSS, and JavaScript.

## Why this is a good first project

- Easy to understand: one HTML file contains the whole game.
- Easy to publish: upload it to GitHub and enable GitHub Pages.
- Easy to improve: add enemies, power-ups, sound, levels, or mobile controls.

## How to run

1. Download or clone the project.
2. Open `neon-dash-rescue.html` in your browser.
3. Or upload the folder to GitHub and serve it with GitHub Pages.

## Controls

- Move: `WASD` or arrow keys
- Pause: `P` or the pause button
- Start: click the button

## Simple upgrade ideas

- Add a start menu with difficulty levels.
- Add touch controls for phones.
- Add combo multipliers and boss waves.
- Split the code into `index.html`, `style.css`, and `game.js` later.
